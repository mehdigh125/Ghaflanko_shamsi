# Description
Enter the Gregorian date as an input and  convert to the hijri date
# Ghaflanko_shamsi
+ Input :
<br>
    year=1986
<br>
    month=8
<br>
    day=18
<br>
    Ghaflanko_shamsi.today(year,month,day)
<br><br><br>

+ Output :
<br>
    1365/05/27


# Installation 

```bash
pip Ghaflanko_shamsi
```
